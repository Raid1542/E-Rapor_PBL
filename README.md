Aplikasi E-Rapor SDIT Ulil Albab Batam adalah sistem informasi akademik berbasis web yang dirancang untuk membantu guru, wali kelas, dan admin dalam mengelola nilai siswa secara digital, efisien, dan aman.

### ğŸ”§ Fitur Utama:
- Input nilai harian, UTS, dan UAS oleh Guru Bidang Studi.
- Input catatan wali kelas, absensi, dan penilaian ekstrakurikuler oleh Wali Kelas.
- Manajemen data siswa, guru, kelas, dan mata pelajaran oleh Admin.
- Impor data nilai dari Excel untuk menjaga kebiasaan kerja guru.
- Cetak rapor otomatis dalam format PDF.
- Arsip rapor digital untuk alumni.
- Pembagian hak akses multi-role (Admin, Wali Kelas, Guru Bidang Studi).
